DDoS 的攻击和防御

day 1

前言

说实话 chatGPT 讲得更好（不是）

DDoS，全称为分布式拒绝服务（Distributed Denial of Service），是一种恶意攻击方式，旨在通过超载目标系统的网络、服务器或应用程序，使其无法正常提供服务。下面是有关 DDoS 攻击的一些基本信息：

  1. 攻击原理：

    • 单个攻击者很难通过自己的计算机或网络连接对目标系统进行有效的攻击。因此，攻击者会利用分布式网络或僵尸网络（botnet），其中包含大量被感染或控制的计算机和设备。
    • 攻击者使用这些被感染的计算机/设备作为“僵尸”，通过将它们的带宽和计算资源联合起来，协调并向目标系统发送大量的请求流量。
    • 这些请求流量可能是无效的、重复的或具有高度计算密集性的请求，导致目标系统耗尽资源或无法正常处理合法用户的请求。求。

  2. 攻击类型：

    • 带宽攻击：这种攻击类型旨在耗尽目标系统的带宽，使其无法处理合法流量。攻击者发送大量的数据流量，以超过目标系统的处理能力。
    • 连接数攻击：攻击者尝试通过建立大量的连接请求来占用目标系统的资源，例如 TCP 半连接攻击（SYN Flood）或连接保持攻击。这样做会导致目标系统在处理这些连接时变得资源吃紧。
    • 应用层攻击：这种攻击目标是系统的应用层资源，例如 Web 服务器或数据库。攻击者可能发送大量的 HTTP 请求、恶意脚本或利用系统漏洞来导致目标系统处理不了这些请求。

  3. 攻击工具和手段：

    • 僵尸网络（botnet）：攻击者通常会通过各种方式感染大量计算机或设备，将其组织成一个大规模的僵尸网络。攻击者通过控制这个僵尸网络，可以协调并发动 DDoS 攻击。
    • 拒绝服务攻击工具包（DDoS toolkits）：这些工具包可供攻击者使用，方便执行 DDoS 攻击。它们提供了各种攻击类型的功能，包括生成大量恶意流量、伪造源 IP 地址等功能。
    • 资源放大攻击：这是一种利用某些网络协议或服务的漏洞，向目标系统发送少量请求，但在目标系统上触发大量响应的攻击方式。这样可以放大攻击者的流量，使得攻击更具破坏性。

  4. 影响：

    • 服务不可用：DDoS 攻击会导致目标系统无法正常提供服务，造成服务中断、网站崩溃或缓慢响应等问题。
    • 业务损失：由于服务不可用，企业可能会遭受停机损失、业务中断、客户流失等经济损失。
    • 威胁网络安全：DDoS 攻击也可能被用作分散注意力、掩盖其他攻击或滥用系统漏洞的手段。

  5. 防御措施：

    • 流量清洗和 DDoS 防护：组织可以使用专业的 DDoS 防护服务来过滤和清洗流量。这些服务会检测和分析流量，并识别并隔离恶意流量，以确保只有合法的流量到达目标系统。
    • 弹性扩展：通过使用云计算服务，可以在受到攻击时快速增加服务器资源来应对大量的请求流量。负载均衡技术也可以帮助分散流量，减轻单个服务器的负担。
    • 防火墙规则和黑名单：配置防火墙规则、IP 地址黑名单和白名单，以限制恶意流量的访问。这些规则可以根据特定的攻击模式和来源 IP 地址进行过滤，从而减轻攻击对目标系统的影响。
    • 实时监测和响应：建立实时监测系统，及时发现流量异常或攻击行为，并采取相应的响应措施。这可能包括封锁恶意 IP 地址、更新防火墙规则、通知 ISP（互联网服务提供商）来协助应对攻击等。

  6. 合规要求：

    • 根据中国相关法律法规，必须遵守网络安全法、计算机信息系统安全保护条例等规定。组织应确保其防御措施符合相关的合规要求，包括数据保护、用户隐私和网络安全等方面。

  7. 需要指出的是 DDoS 攻击技术不断发展，攻击者的手段也在不断演变。因此，组织在保护自身免受 DDoS 攻击时，需要时刻保持警惕，及时更新和改进其防御策略和措施，以确保网络和系统的安全。总体而言，DDoS 攻击是一种严重的网络安全威胁。为了保护目标系统免受此类攻击，安全防护措施、网络监测和实时响应是至关重要的。

一、DDoS 的定义

DDoS 全称 Distributed Denial of Service，意思为“分布式拒绝服务”，即利用大量合法的分布式服务器对目标发送请求，从而导致正常合法用户无法获得服务。恶意行为。

二、DDoS 的特点

  1. 造成网络瘫痪
    通过控制大量攻击主机同时连接网站端口，制造大流量无用数据，造成通往被攻击主机的网络拥塞，使被攻击主机无法正常和外界通信。

  2. 肉鸡攻击
    肉鸡也称傀儡机，是指可以被黑客远程控制的机器，肉鸡通常被用作 DDoS 攻击，通过肉鸡发起的攻击请求占用服务器各项资源。

  3. 网络层攻击
    通过发送伪造源 IP 的 TCP 数据包发送 SYN 或 ACK 包、发送包含错误设置的地址值等攻击方式，耗尽服务器资源，导致服务器拒绝访问。

  4. CC 攻击
    黑客借助代理服务器生成指向受害主机的合法网页请求，实现 DDoS 和伪装就叫 CC（Challenge Collapsar）。肉鸡 CC 攻击是黑客使用 CC 攻击软件，控制大量肉鸡发动攻击，模拟正常用户访问网站的请求，伪造成合法数据包。

三、DDoS 的分类

（一）自动化程度分类

  1. 手动攻击
  2. 半自动化攻击
  3. 自动化攻击

（二）协议分类

  1. 洪水攻击 udp（使之带宽饱和）
  2. 利用协议的攻击
  3. 利用广播 ip 的攻击
  4. 利用数据包攻击（IP 地址攻击和 IP 数据包属性攻击）

（三）速率分类

  1. 持续速率 影响快
  2. 可变速率

（四）影响力分类

  1. 网络服务彻底崩溃
  2. 网络服务能力降低

（五）入侵目标

  1. 带宽攻击
  2. 连通性攻击

四、被 DDoS 的表现

被攻击主机上有大量求情

充斥大量 TCP 数据包
制造大量无用数据 导致网络拥塞 无法与外界网络连接
