计算机网络基础分类

发表于2023-09-03|更新于2023-09-04
|阅读量:

一、网络概述

（一）交换机

交换机（Switch）：

  • 交换机是一种网络设备，用于在局域网内部进行数据包转发和交换。
  • 交换机基于 MAC 地址（物理地址）来识别目标设备，并直接将数据包转发到相应的端口，实现设备间的直接通信。
  • 交换机通常用于构建局域网，连接各个设备（如计算机、服务器、打印机等）以实现高速、可靠的内部通信。
  • 交换机具有多个端口，可以同时与多个设备进行通信，实现数据的快速传输和广播。

（二）路由器和防火墙

1）路由器（Router）：

  • 路由器是一种网络设备，用于在不同的网络之间进行数据包转发和路由选择（数据处理）。
  • 路由器通过查看目标 IP 地址，并根据路由表中的信息决定如何传输数据包。
  • 路由器连接不同的网络，并起到网络互连的作用，例如将局域网（LAN）与广域网（WAN）连接起来。
  • 路由器还具备网络地址转换（NAT）的功能，可以将私有 IP 地址转换为公有 IP 地址，实现多个设备共享一个公网 IP 地址。

2）防火墙（Firewall）

一种网络安全设备或软件，用于监控和控制网络流量，以保护计算机网络免受未经授权的访问、恶意攻击和网络威胁。
（路由器+防毒模块）
防火墙的主要功能包括：

  1. 包过滤（Packet Filtering）：
    • 防火墙通过检查网络数据包的源地址、目标地址、端口号等信息，并根据预先设置的规则决定是否允许通过。
    • 这些规则可以基于协议类型（如 TCP、UDP）、IP 地址、端口等条件进行配置。
  2. 访问控制列表（Access Control List，ACL）：
    • 防火墙可以使用 ACL 来定义允许或禁止特定主机或网络对网络资源的访问权限。
    • ACL 可以基于源 IP 地址、目标 IP 地址、端口等参数来限制对网络服务的访问。
  3. 状态检测（Stateful Inspection）：
    • 防火墙可以维护有关网络连接状态的信息，识别并允许通过与已建立的合法通信相关的回复数据包。
    • 这种状态检测可以提高防火墙的安全性，防止某些攻击类型，如欺骗、会话劫持等。
  4. 应用层代理（Application Layer Proxy）：
    • 一些高级防火墙可以提供应用层代理，对特定应用协议的流量进行深度检查和控制。
    • 这种防火墙能够检测和阻止潜在的网络攻击、恶意软件和不良内容。
  5. 虚拟专用网络（Virtual Private Network，VPN）： - 部分防火墙设备或软件支持 VPN 功能，通过加密和隧道技术创建安全的远程连接，保护数据在公共网络上的传输安全性。
    通过配置和管理防火墙，组织可以限制网络访问、防范网络攻击，并保护关键数据和资源的安全性。防火墙通常成为企业和个人网络安全架构中重要的一环。

（三）AP、AC

1）AP（Access Point）：

  • AP 是无线接入点的缩写，也称为无线基站或无线路由器。
  • AP 负责发射和接收无线信号，提供无线网络覆盖范围。
  • 当无线客户端设备（如笔记本电脑、智能手机）与 AP 进行无线连接时，AP 会将数据包从有线网络转发到无线网络，或者将无线数据包转发到有线网络。
  • 在大型网络环境中，可能需要多个 AP 来覆盖更广阔的区域。

2） AC（Access Controller）：

  • AC 是接入控制器的缩写，也称为无线控制器。
  • AC 是用于集中管理和控制多个 AP 的设备。
  • AC 负责配置和监控 AP 的工作状态，为无线客户端设备提供统一的认证、安全策略和流量调度。
  • 通过 AC，管理员可以集中管理所有 AP 的配置，例如无线网络名称（SSID）、加密方式、访问控制等。
  • AC 还可以对无线用户进行身份验证、按需分配网络资源，并提供对无线网络的流量管理和优化。

（四）WEB 应用防火墙（Next Generation Firewall，NGFW）

传统防火墙=路由器 PRO （具有一部分杀毒功能）

下一代防火墙 （魔力象限提出的概念）
概念：多产品融合 = ips ids waf 防火墙

对传统防火墙的进一步演进和提升，它融合了传统防火墙的功能以及更多高级的安全特性和智能功能。下一代防火墙在网络安全中扮演着重要的角色，以下是一些与下一代防火墙相关的特点和功能：

  1. 应用识别与控制：下一代防火墙能够深度检测网络流量中的应用程序，可以识别和控制各种应用层协议。它能够对不同的应用进行策略控制，限制或允许特定应用的使用。
  2. 用户识别和访问控制：下一代防火墙可以识别特定用户或用户组，并根据用户身份、角色和权限设置访问控制策略。这样可以实现更精细的用户级别访问控制，提高网络安全性。
  3. 内容过滤和恶意软件防护：下一代防火墙可以检测和阻止网络流量中的恶意软件、病毒、间谍软件等恶意内容。它利用先进的威胁情报和行为分析技术，提供实时保护和防御。
  4. 入侵检测与防御系统（Intrusion Detection and Prevention System，IDPS）：下一代防火墙集成了 IDPS 功能，能够检测和阻止网络中的各类入侵行为，包括漏洞利用、DoS（拒绝服务）攻击等。
  5. SSL/TLS 解密与审计：下一代防火墙能够解密和审计通过 SSL/TLS 加密的流量，这样可以进行更全面的流量分析和检测。
  6. 高级威胁防护（Advanced Threat Protection，ATP）：下一代防火墙通过整合多种安全技术，如行为分析、沙盒检测等，提供高级威胁的检测和防范能力。
  7. 网络可视化和报告：下一代防火墙提供实时的网络流量可视化和安全报告，管理员可以清晰地了解网络中的流量状况、安全事件和威胁情报，以便做出相应的安全决策。

（五）上网行为管理

指对组织或网络环境中的用户的上网活动进行监控、控制和管理的过程。通过上网行为管理，组织可以加强网络安全、优化网络资源利用以及遵守法规和政策要求。下面是一些常见的上网行为管理措施：

  1. 访问控制：通过设置访问控制策略，限制特定用户或用户组对某些网站、应用程序或特定网络资源的访问权限。这可以通过基于角色、身份验证、IP 地址、时间段等来实现。
  2. 内容过滤：使用内容过滤技术来阻止或限制特定类型的网站、应用程序或内容的访问。这包括恶意软件、色情、赌博、社交媒体等不良内容。
  3. 带宽管理：通过带宽管理技术，对网络的带宽进行合理分配和控制，以确保关键应用程序和任务的稳定运行，并限制非业务相关的带宽消耗。
  4. 应用程序控制：对特定的应用程序进行识别和控制，可以限制某些应用程序的使用或优化关键应用程序的性能，以确保网络资源的合理利用。
  5. 安全审计和日志记录：通过安全审计和日志记录技术，记录用户的上网活动和访问行为，以便进行安全审计、威胁检测和事件响应。
  6. 行为分析和威胁检测：利用行为分析和威胁检测技术，监测异常的网络活动和潜在威胁，及时发现并采取措施进行防范。
  7. 用户教育与培训：通过向用户提供网络安全教育和培训，增强他们对安全风险的认识，并促使他们遵守组织制定的上网行为规范。

（六）核心安全产品 AF

安全即服务 SaaS

AF SIP EDR（Advanced Firewall, Security Information and Event Management, Endpoint Detection and Response）是一种核心安全产品，结合了高级防火墙、安全信息和事件管理以及终端检测与响应的功能。

  1. 高级防火墙（Advanced Firewall）：提供强大的网络流量过滤和访问控制功能，能够有效地保护企业网络免受未经授权的访问和恶意攻击。
  2. 安全信息和事件管理（Security Information and Event Management）：监控和分析网络中的安全事件和日志，将来自不同系统和设备的安全事件集中管理，帮助快速识别、报告和响应潜在的安全威胁。
  3. 终端检测与响应（Endpoint Detection and Response）：在终端设备上部署专门的安全代理或软件，实时监测终端活动并检测异常行为，及时发现和应对潜在的恶意活动和安全威胁。

AF SIP EDR 综合了网络层、主机层和安全信息管理层的安全能力，提供全面的威胁检测和响应能力。它可以：

  • 检测和防御网络入侵、恶意软件和其他安全威胁。
  • 监控和分析网络流量和终端活动，提供实时的安全事件响应和报告。
  • 收集、分析和聚合来自各种安全设备和系统的日志数据，以便进行整体威胁情报和安全分析。
  • 提供对网络和终端设备的远程管理和配置，以及快速响应和修复漏洞和安全问题。
